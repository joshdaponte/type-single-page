# Fraunces – Variable Font Website

## Overview
This project is a responsive website that explores the Fraunces variable font. The goal was to showcase its history, type characteristics, and unique variable axes such as weight, softness, and wonk.

## Objective
The project aims to communicate the personality of Fraunces through an interactive and visually engaging poster-styled layout.

## Features
- Responsive layout for desktop and mobile
- Sections for history, variations, and type exploration
- Website styled to compliment the typeface design

## Solution
This project demonstrates how variable fonts can be presented clearly and interactively on the web.

## Learned
- Strengthened my ability to implement responsive design for mobile devices.
- How variable font properties can be explored visually, and animating them in the way I want.

## Project
The design focuses heavily on typography as the primary visual system, with the use of color and texture to compliment it based on its meaning and history.

## Fonts
- Fraunces

## Accessibility
- Color contrast for readability
- Responsive design for different screen sizes

## Design Process
I designed the entire site in Figma, where I laid out the type, colors, and spacing. Then I translated that design into HTML and CSS, trying to maintain accuracy between my initial mockup and the live website.
